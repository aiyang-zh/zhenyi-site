# zhenyi-site

Zhenyi 官网，高性能实时技术三层后端生态展示页。

**官网地址**：<https://zhenyi-site.pages.dev/>

## 内容

- 三层生态介绍（MIT 基础库 · AGPL 框架 · 商业套件）
- 性能基准图表（Network Echo 压测、核心组件）
- 合作方式（开源贡献、商业合作、安全反馈）

## 本地预览

```bash
# 使用 Python 启动本地服务
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 部署

部署至 Cloudflare Pages，构建命令留空（静态站点），输出目录为根目录。
